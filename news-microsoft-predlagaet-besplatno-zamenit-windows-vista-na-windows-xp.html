Microsoft запускает ранее анонсированную программу бесплатного обмена потребителями операционной системы Windows Vista Home Basic и Windows Vista Home Premium на Windows XP.

Это еще один шаг навстречу российским пользователям, призванный повысить их удовлетворенность от использования лицензионного программного обеспечения Microsoft. Предложение разработано специально для России и будет действовать до конца 2009 года.
