Про то,  как настроить мультитач на windows-ноутбуках я уже писал. Организованно это было с помощью хорошей бесплатной программы TwoFinferScroll.  Недавно на сайте производителе ноутбуков HP появилась новая версия драйверов для тачпадов, в настройках которых включены опции двух- и трехпальцевого нажатия и прокрутки.

Скачать драйвера можно по ссылкам:

- для Windows XP

- для Windows Vista\ Seven (7)

После установки драйверов необходимо перезагрузить ноутбук. В настройках тачпада будут доступны следующие новые функции:

- настройки вертикальной и горизонтальной прокрутки двумя пальцами

Для избежания путаницы на тачпадах небольшого размера прокрутку одним пальцем можно отключить.

- нажатие тремя пальцами

Появится возможность запуска по нажатию по тачпаду тремя пальцами любого приложения (или даже нескольких приложений).

Самое частоиспользуемое приложение можно поместить в этот список и запускать его в один клик.

- настройка пролистывания тремя пальцами в различных приложениях (например, интернет-браузере)

- так же можно настроить чувствительность, зоны легкого касания пальцем и многое другое.

В дополнение к вышеуказанным настройкам можно использовать уже известную программу TwoFinferScroll. , и настроить клик по тачпадудвумя пальцами, либо комбинацией пальцев.

Таким образом, можно добиться настройки управления тачпадом  наиболее удобного для себя.

Дерзайте, и все получится!
